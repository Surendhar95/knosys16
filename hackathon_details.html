Are you crazy about cracking things? How about exploiting? Can you hack? Here we are with an event to challenge your hacking ability!

Event Format

  1. Each group is given a system with MSF console.

  2. The team is given a target IP address (Windows XP System).

  3. The main objective of this event is to exploit the target and complete the tasks given

Rules

  1. Time Duration 1 Hour

  2. Two Members per group

  3. Using internet is allowed

  4. Team which completes first will be considered as winners

  5. Team can bring their own laptop(Recommended)

  6. The Complete event will be organised in 2 Levels

Pre-requisites

  1. Basic computer networking

  2. Metasploit framework

  3. Windows and Linux command line
