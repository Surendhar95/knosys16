This course is desgined to provide you with the tools and techniques used by hackers and information security professionals alike to break into an organization. As we say it,“To beat a hacker, you need to think like a hacker”. This course will immerse you into the Hacker's Mindset so that you will be able to defend against future attacks. It puts you on the driver’s seat of a hands-on environment with a systematic ethical hacking process. The goal of this course is to help you master ethical hacking methodologies that can be used in a penetration testing or ethical hacking situation. You walk out with ethical hacking skills that are highly in demand.


Course Content

HackPro-( Ethical Hacker and Counter Measures Expert).

This workshop consists of 15 core modules designed to facilitate a comprehensive ethical hacking and penetration testing training.

  1. Introduction to Ethical Hacking

  2. Footprinting and Reconnaissance

  3. Scanning Networks & Enumeration

  4. System Hacking

  5. Malware Threats

  6. Sniffing

  7. Social Engineering

  8. Denial of Service

  9. Session Hijacking

  10. Hacking Web Servers

  11. Hacking Web Applications

  12. SQL Injection

  13. Hacking Wireless Networks

  14. Hacking Mobile Platforms

  15. Evading IDS,Firewalls,and Honeypot


What will you learn??


  1. Key issues plaguing the information security world, incident management process, and penetration testing

  2. Various types of footprinting, footprinting tools, and countermeasures

  3. Network scanning techniques and scanning countermeasures

  4. System hacking methodology, steganography, steganalysis attacks, and covering tracks

  5. Different types of Trojans, Trojan analysis, and Trojan countermeasures.

  6. Working of viruses, virus analysis, computer worms, malware analysis procedure, and countermeasures

  7. Packet sniffing techniques and how to defend against sniffing

  8. Social Engineering techniques, identify theft, and social engineering countermeasures

  9. DoS/DDoS attack techniques, botnets, DDoS attack tools, and DoS/DDoS countermeasures

  10. Session hijacking techniques and countermeasures

  11. Different types of webserver attacks, attack methodology, and countermeasures

  12. Different types of web application attacks, web application hacking methodology, and countermeasures

  13. SQL injection attacks and injection detection tools

  14. Wireless Encryption, wireless hacking methodology, wireless hacking tools, and wi-fi security tools

  15. Mobile platform attack vector, android vulnerabilities, jailbreaking iOS, windows phone 8 vulnerabilities, mobile security guidelines, and tools

  16. Firewall, IDS and honeypot evasion techniques, evasion tools, and countermeasures
