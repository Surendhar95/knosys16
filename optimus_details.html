OPTIMUS

Think you know the difference between a good code and great code? Have the thing in you to turn a good code into a great code? Then, this is your event.
Tap your feet to the rhythm of efficiency!!!


Event Format

  1. The event consists of single round.

  2. This is an individual event.

  3. The participant has to optimize a given code in maximum number of ways possible (eg. space, time) and show the output.

  4. The output format of the optimized code must be same as that of the code given initially.

  5. Questions will be in C,C++ .

  6. Questions will be in increasing order of difficulty and so are the marks.

  7. Operating system will be Ubuntu 13.04.Platform will be GCC Compiler. No IDEs will be provided.


Rules

  1. Individual participant has to register for the event.

  2. The time limit for the event will be 1 hour and 30 minutes

  3. The number and nature of problems and the evaluation criteria will be revealed along with the problem statements, at the venue.

  4. Use of mobile phones or other electronic devices will not be permitted.

  5. Participants must carry their respective college identity proof.

  6. Any kind of unfair means will lead to disqualification.

  7. The judges reserve right to disqualify if any discrepency found.
